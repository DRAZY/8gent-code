## 8gent Onboarding Guide

### Introduction
8gent is an autonomous AI coding agent that helps with development tasks. This guide covers setup and usage.

### Prerequisites
- Node.js 18+
- Git installed

### Getting Started
1. Clone your project
2. Run `npx 8gent`
3. Use `Ctrl+T` to switch modes

### Basic Commands
- `code` - Write code
- `debug` - Diagnose issues
- `test` - Run tests

### Example Workflow
```bash
8gent: code a TypeScript API
8gent: test the implementation
8gent: debug any failures
```

### Troubleshooting
- If commands fail: try alternative approaches
- For errors: use `web_search` to find solutions
- Complex tasks: spawn specialist agents

### Advanced Features
- Use `suggest_spawn` for complex tasks
- Leverage design system with `suggest_design`
- Run background processes with `background_start`
