# 8gent Brand Values

## Core Values

### 1. Freedom Over Convenience
Cloud agents are convenient — until you hit the cap, until the price goes up, until they deprecate your favorite model. 8gent chooses freedom: local execution, model choice, no vendor lock-in. Convenience fades; freedom compounds.

### 2. Methodology Over Brute Force
Claude Code and Codex rely on raw model intelligence to get things right. That works great — until it doesn't. 8gent pairs model intelligence with structured methodology (BMAD). Planning, execution, verification, learning. Structure beats improvisation over time.

### 3. Privacy as a Right, Not a Feature
Your code is your IP. It shouldn't stream to someone else's servers as the cost of getting AI help. 8gent runs locally. Your code stays on your machine. Period. This isn't a premium feature — it's the default.

### 4. Developer Delight
The terminal is where serious work happens. It shouldn't feel like a punishment. 8gent brings rich UI, animations, voice, ADHD-friendly modes, and thoughtful design to the CLI. Other agents give you a bare prompt. We give you an experience.

### 5. Self-Improvement
Cloud agents improve when the vendor ships an update. 8gent improves itself — autoresearch, benchmarking, RL fine-tuning via the kernel. The agent learns from its own performance and gets better with use.

### 6. Radical Openness
MIT licensed. Fully open source. No telemetry you didn't opt into. No "open core" bait-and-switch. The community can inspect, modify, fork, and extend everything. Trust is earned through transparency.

### 7. Craftsmanship
Every interaction, every animation, every error message is intentional. We take pride in code quality and user experience because developers notice the difference between "built to ship" and "built to last."
