# 8gent Brand Discovery Workshop Agenda

## Workshop Overview
This workshop is designed to define and refine the brand identity for 8gent, the autonomous AI coding agent. The goal is to create a cohesive, compelling brand that resonates with developers and positions 8gent as the premier local LLM-powered coding assistant.

## Duration
1 Full Day (8 hours)

## Participants
- Founder/Product Lead (James Spalding)
- Lead Developer
- Designer (if available)
- Community Representative (optional)
- Facilitator (8gent AI Agent)

## Agenda

### 1. Introduction & Icebreaker (9:00 - 9:30)
- Welcome and objectives
- Quick introductions
- Icebreaker: "If 8gent were a historical figure, who would it be and why?"

### 2. Brand Foundations (9:30 - 10:30)
- Exercise 1: Brand Heart (Why, How, What)
- Exercise 2: Brand Values
- Exercise 3: Brand Personality

### 3. Audience & Market Analysis (10:30 - 12:00)
- Exercise 4: User Personas
- Exercise 5: Competitive Analysis
- Exercise 6: Market Positioning

### 4. Visual Identity Exploration (13:00 - 14:30)
- Exercise 7: Logo Concepts
- Exercise 8: Color Palette
- Exercise 9: Typography
- Exercise 10: Imagery & Iconography

### 5. Brand Voice & Messaging (14:30 - 15:30)
- Exercise 11: Brand Voice Principles
- Exercise 12: Messaging Framework
- Exercise 13: Tagline Refinement

### 6. Brand Applications (15:30 - 16:30)
- Exercise 14: TUI Experience
- Exercise 15: Web Interface
- Exercise 16: Merchandise
- Exercise 17: Documentation & Community

### 7. Brand Guidelines & Next Steps (16:30 - 17:00)
- Exercise 18: Brand Guidelines Overview
- Exercise 19: Implementation Roadmap
- Closing and next steps

## Deliverables
- Brand Heart Document
- Brand Personality Matrix
- User Personas
- Competitive Positioning Map
- Logo Concepts (3 directions)
- Color Palette & Typography Guide
- Brand Voice Chart
- Messaging Framework
- Application Guidelines (TUI, Web, Merch)
- Brand Guidelines Document
- 30-60-90 Day Implementation Plan

## Materials Needed
- Whiteboard or large paper
- Markers
- Sticky notes
- Printed exercises
- Laptop for research
- 8gent TUI for live demo

## Success Criteria
- Clear, differentiated brand positioning **relative to Claude Code specifically**
- Cohesive visual and verbal identity
- Actionable implementation plan
- Team alignment on brand direction
- Honest competitive analysis that acknowledges where we're behind (model quality, brand recognition) and where we win (freedom, privacy, experience, methodology)

## Pre-Workshop Required Reading
- Updated COMPETITIVE_ANALYSIS.md (Claude Code as primary competitor)
- Browse r/ClaudeAI, r/LocalLLaMA, and Claude Code GitHub issues for real user pain points
- Try Claude Code for 1 hour and note what's great and what frustrates you
- Try Aider + Ollama for 1 hour as the closest OSS comparable
