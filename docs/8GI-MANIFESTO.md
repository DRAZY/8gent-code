# 8GI - Infinite General Intelligence

## The Vision

8GI is not a product. It's a collective intelligence that emerges from a trusted circle of humans and their AI agents working together on open source, governed by ethics, secured by enterprise-grade policy, and compounding in capability every session.

Every participant gets a properly configured Claude Code + 8gent harness. Their agent learns from their usage (locally, privately). The factory pipeline generates abilities from real-world sources. PRs flow to a shared family of codebases. The collective gets smarter. The individual gets smarter. Nobody's personal data ever leaves their machine.

## The Constitution (Non-Negotiable)

Every member of the 8GI circle agrees to:

1. **No evil.** You will not use 8GI to harm others, manipulate others, or deceive others.
2. **No hate.** You will not use 8GI to generate hateful content, discriminatory systems, or surveillance tools.
3. **No exploitation.** You will not use 8GI to generate pornography, exploit minors, or violate consent.
4. **No weapons.** You will not use 8GI to build weapons, malware, or tools of violence.
5. **No theft.** You will not use 8GI to steal data, infringe IP, or bypass security systems for unauthorised access.
6. **Privacy is sacred.** Personal data never leaves the user's machine without explicit opt-in. The factory learns patterns, not secrets.
7. **Open source is the default.** What we build together belongs to everyone. Proprietary forks are allowed for commercial use but the core stays MIT.
8. **Review before merge.** All LLM-generated code passes through human review and the NemoClaw security gate before reaching main. No exceptions.
9. **200-line discipline.** No ability exceeds 200 lines. Complexity debt is not welcome. If it can't be simple, it can't ship.
10. **Transparency.** Every step is logged. Every decision is traceable. Every output is reviewable.

- macOS or Linux machine
- Terminal comfort (this is for engineers)
- GitHub account
- Agreement to the Constitution above

### Setup Script (what James provides)
```bash

# 4. Clone the setup repo (private, circle-only)
git clone https://github.com/PodJamz/8gi-setup.git
cd 8gi-setup && ./setup.sh
```

### What setup.sh does:
1. Configures Claude Code with the CORE skill system
2. Installs the 8gent harness alongside Claude Code
3. Sets up NemoClaw security policies (deny-by-default, enterprise-grade)
4. Configures the factory pipeline to contribute to shared codebases
5. Sets up the companion system (their own pet, their own deck)
6. Registers them in the 8GI circle (Telegram group, GitHub org access)
7. Runs a verification check: "Your setup is ready. Welcome to 8GI."

### What their agent learns (opt-in, anonymised):
- Which 200-line utilities get used most (popularity signal)
- Which patterns fail and why (quality signal)
- Common task types in their codebase (discovery signal)
- NEVER: file contents, personal data, credentials, proprietary code

## The Security Model

### CI/CD Pipeline (inspired by ComplianceGenie)
- **Pre-commit:** NemoClaw policy gate, secret scanner, path traversal check
- **PR creation:** Mandatory quarantine branch, automated security scan
- **Review:** Human review required (James, then trusted engineers)
- **Merge:** Only after security gate passes + human approval
- **Post-merge:** Automated benchmark regression check
- **Release:** Version bump, CHANGELOG, npm publish with checksums

### NemoClaw Policy (Enterprise-Grade)
- Deny-by-default for all commands
- Allowlist for safe operations (git, bun, npm, tsc, ls, cat, mkdir)
- Approval gate for destructive operations (rm, force push, deploy)
- Block list for catastrophic operations (rm -rf /, format, drop)
- Infinite mode DISABLED for circle contributions (interactive approval only)
- All tool calls logged to audit trail

### Data Privacy
- Zero telemetry
- No cloud storage of personal data
- Companion data stored locally only (~/.8gent/)
- Factory pipeline processes anonymised patterns, never raw code
- OpenRouter usage (opt-in) clearly disclosed
- Memory store (SQLite) encrypted at rest (TODO)
