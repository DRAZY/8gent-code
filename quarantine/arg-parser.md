# arg-parser

Minimal CLI argument parser with typed flags, positional args, and help generation.

## Requirements
- define flags with type ('string'|'boolean'|'number'), default, alias
- parse(argv) returns typed result object
- Positional arguments collected in _
- generateHelp(programName) -> string
- Unknown flag error with did-you-mean suggestion
