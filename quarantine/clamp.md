# clamp

Math utilities: clamp, lerp, remap, wrap, and smooth-step.

## Requirements
- clamp(value, min, max) -> number
- lerp(a, b, t) -> number linear interpolation
- remap(value, inMin, inMax, outMin, outMax) -> number
- wrap(value, min, max) -> number wrapping modulo
- smoothstep(edge0, edge1, x) -> number
