# test-runner

Minimal test runner with describe/it/beforeEach/afterEach.

## Requirements
- describe(name, fn), it(name, fn), test(name, fn)
- beforeEach(fn), afterEach(fn), beforeAll(fn), afterAll(fn)
- Run tests with run() -> { passed, failed, duration }
- Async test support
- Minimal output format: pass/fail with timing
