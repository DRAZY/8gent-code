# bit-flags

Type-safe bit flag utilities for permission/option sets.

## Requirements
- createFlags(names[]) -> typed flag object
- combine(...flags) -> number
- has(value, flag) -> boolean
- toggle(value, flag) -> number
- toNames(value, flags) -> string[]
