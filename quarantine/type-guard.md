# type-guard

Runtime type guards and assertion utilities.

## Requirements
- isString, isNumber, isBoolean, isNull, isUndefined, isArray, isObject
- isNonNull(val): val is NonNullable<T>
- assert(condition, message) throws if false
- assertNonNull(val, message)
- isRecord(val) -> val is Record<string, unknown>
