Emergency Powers

Speed when it matters. Accountability always.

Founder Unilateral Action

The Founder can act without prior vote in three specific scenarios:

  • Security vulnerabilities - active exploits, credential leaks, supply chain attacks
  • Member violations - immediate harm to users, data breaches, malicious code
  • Infrastructure failure - service outages requiring immediate architectural changes

Safeguards

  • All emergency actions must be ratified by Core within 7 days
  • Every action is logged with timestamp, rationale, and impact
  • Logs are visible to all Core members
  • If Core does not ratify, the action must be reversed

Transparency Principle

Emergency powers exist for speed, not secrecy. The circle sees everything. Trust is maintained through radical transparency, not authority.

Lotus Scaling Model

4 concentric rings. Governance expands outward as the circle grows.

Ring 0 - Founder Decides

Seed phase. Single decision-maker. Maximum speed, full accountability. All strategic and operational calls flow through one person.

NOW

Ring 1 - Chiefs Vote

Inner circle. 8 seats maximum. Chiefs vote on strategic direction, resource allocation, and cross-domain decisions. Majority rules, Founder breaks ties.

NEXT

Ring 2 - Heads Operate (BMAD)

Heads run through BMAD phases: Analysis - Planning - Solutioning - Implementation. Each phase has a review gate before progression.

LATER

Ring 3 - Leads Run Sprints

Sprint leads execute within defined scope. Review gates at sprint boundaries. Leads report up to Heads, never skip rings.

SCALE

Three Mandatory Gates

Every transition between rings passes through all three gates. No exceptions.

NemoClaw Gate
Security review. Policy engine validates all changes against deny-by-default rules.
Alignment Gate
Ethics review. Constitutional compliance. Values check against the 8gent Constitution.
Quality Gate
Design review. UX, accessibility, and brand compliance. No shortcuts on craft.

The Lotus model scales without losing control. Inner rings set direction, outer rings execute. Gates enforce standards at every boundary.
