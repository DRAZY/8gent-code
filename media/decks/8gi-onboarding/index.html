The Constitution

10 rules. Non-negotiable. Read the full version before you agree.

  • No evil. Don't use this to harm, manipulate, or deceive people.
  • No hate. No discriminatory systems, surveillance tools, or hateful content.
  • No exploitation. No consent violations. No exploiting anyone.
  • No weapons. No malware, no tools of violence.
  • No theft. No data theft, no IP infringement, no unauthorized access.
  • Privacy is sacred. Personal data never leaves your machine without explicit opt-in.
  • Open source is the default. The core stays MIT. What we build together belongs to everyone.
  • Review before merge. All LLM-generated code goes through human review + NemoClaw. No exceptions.
  • 200-line discipline. No ability exceeds 200 lines. If it can't be simple, it can't ship.
  • Transparency. Every step logged, every decision traceable, every output reviewable.

Your First PR

The loop: you code, the agent helps, the PR lands, the collective improves.

01 Pick Find a good-first-issue on GitHub. Start small - fix a typo, improve a test, clean up a comment.
02 Branch Create a quarantine branch: quarantine/{you}/{feature}. All new code starts untrusted.
03 Build Use Claude Code as your pair programmer. NemoClaw enforces policies as you work. 200-line max per file.
04 Test Run bun run tui and verify it works. Never push untested code. Tests or it didn't happen.
05 CI Pipeline Security scan, type check, lint, test suite, dependency audit. All automated, all required to pass.
06 Human Review James reviews all PRs initially. As trust builds, Core Circle members gain review access.
07 Merge Squash merge to main. Branch deleted. CHANGELOG updated. Celebration in Telegram.
